# AI Lab - InfAU Workshop

A hands-on workshop where Bauhaus InfAU staff use AI-assisted coding to build interactive web applications — no prior web development experience needed.

## What's Inside

| Folder | Contents |
|--------|----------|
| `workshop/` | Workshop guide, prompt cards, cheatsheet, and starter data |
| `results/` | Participant projects, published to [GitHub Pages](https://bauhaus-infau.github.io/ai-lab/) |

## Getting Started

See the full workshop guide: [`workshop/README.md`](workshop/README.md)

**Prerequisites:**
1. [VS Code](https://code.visualstudio.com/download)
2. [Claude Pro or Max](https://claude.ai/pricing) subscription
3. [GitHub account](https://github.com/signup) (free)

## Projects

- **A** - Building Explorer: interactive visualization of Weimar building data
- **B** - Solar Position Dashboard: sun path diagram with real-time calculations
- **C** - Evolutionary Algorithm Playground: genetic algorithm for rectangle packing
- **D** - Free Topic: build anything you want with AI-assisted coding
