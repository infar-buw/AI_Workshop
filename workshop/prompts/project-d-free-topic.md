# Project D: Free Topic

## What You'll Build

Anything you want! This is the open-ended option — bring your own idea, your own data, or just experiment with what AI-assisted coding can do.

---

## How to Write Your Starter Prompt

Describe what you want in plain language. Include:

1. **What** the page should do
2. **What it looks like** (layout, colors, style)
3. **What data** it uses, if any (drop a CSV or JSON file into your project folder)

### Template

```
Create a web page that [does what].
It should [look/feel description].
[Optional: It uses this data file: ...]
Make it fullscreen with a clean, modern design.
```

### Example

```
Create a web page with a personal reading tracker. I can add books with
a title and rating. Show them as cards in a grid. Use a warm color scheme.
Store everything in localStorage so it persists. Make it fullscreen.
```

---

## Tips for Scoping

- **Start small.** Get one thing working before adding more.
- **Iterate.** Each follow-up prompt should add one feature.
- **Be specific.** "Add a dark mode toggle in the top-right corner" works better than "make it look nicer."

---

## Ideas If You Need Inspiration

- A personal dashboard (weather, to-do list, quick links)
- A data visualization of something you care about (bring a CSV or JSON)
- A tool for your daily workflow (unit converter, color palette generator, note-taking app)
- A simple game (memory cards, quiz, reaction timer)
- A portfolio or interactive CV page
